Deep Forest

Uma plataforma para criação e treino de classificadores baseados em Deep Learning para identificação de desmatamento. 

Comece agora

Armazenamento de Dados

Armazene e utilize dados de florestamento de todo o Mundo

O Deep Forest armazena dados de florestamento de várias regiões em diversos instantes. 

Treino de Classificadores

Treine classificadores de Deep Learning para detecção de desmatamento 

No Deep Forest, é possível treinar classificadores de deep learning e armazená-los para determinar regiões desmatadas. 

Visualização de Desmatamento

Visualize as áreas de desmatamento segundo a classificação das redes neurais treinadas

Uma vez que os classificadores foram treinados, é possível utilizá-los para visualizar as regiões de desmatamento.
